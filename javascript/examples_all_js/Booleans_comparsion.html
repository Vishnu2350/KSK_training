JavaScript Booleans and Comparison Operators

Booleans

In JavaScript, a boolean is a data type that can only have one of two values: true or false.

Click the button to see boolean examples.

Comparison Operators

Comparison operators are used to compare two values. The result of a comparison is always a boolean value (true or false).

Available Comparison Operators:

  • == : Equal to
  • === : Strict equal to
  • != : Not equal to
  • !== : Strict not equal to
  • < : Less than
  • <= : Less than or equal to
  • > : Greater than
  • >= : Greater than or equal to
